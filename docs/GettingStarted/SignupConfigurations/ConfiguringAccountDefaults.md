# Configuring Account Defaults

Account defaults include the various configurations that are applied once an account is created successfully. For example, it can be KYC needed for Organisation and Individual accounts, or it can be the minimum or maximum number associated with the account as required or any other check like profile check enabled. 

To configure account defaults, navigate to **Administration** > **Settings** > **Account Defaults**. The following screen appears:
![config account details](img/configaccountdetails.png)
Configure following details as per your requirements:

1. **Oragnisation**
	- **Require KYC for Organisation Accounts** - If this option is enabled, organisation accounts can purchase or activate any services, only when their KYC is approved.
	- **Organisation Profile Check** - If this option is enabled, then the organisation accounts needs to complete their profile, Only then will they be able to purchase the services.

2. **Individual**
	- **Require KYC for Individual Accounts** - If this option is enabled, individual accounts can purchase or activate any services, only when their KYC is approved.
	- **Individual Profile Check** - If this option is enabled, then the individual accounts needs to complete their profile, Only then will they be able to purchase the services.

3. **Other Defaults** 
	- **KYC renewal for accounts** -  If this option is enabled, you can select the duration to renew your KYC verification.
	- **Min. Phone Digit** - Enter the minimum phone digit required.
	- **Max. Phone Digit** - Enter the maximum phone digit required.
	- **Custom Heading For Postal Code** - Enter the pin/zip code.
